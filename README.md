# RGSS-rxdata-extractor

## Description

This is a tool for extracting RPG(Role Playing Game) related data that has `.rxdata` or `.rvdata` ext.

## Usage

```shell
bash ./init_dev_env.sh
# edit config.json
python ./extractor.py
```
